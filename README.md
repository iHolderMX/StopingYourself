# StopingYourself

StopingYourself es una aplicación multiplataforma desarrollada en Flutter con un enfoque en la seguridad, autenticación y una experiencia de usuario premium.

## Características Principales

*   **Autenticación Segura**: Sistema de inicio de sesión robusto.
*   **Recuperación de Contraseña**: Flujo clásico mediante envío de código por correo electrónico.
*   **Preparado para 2FA**: Arquitectura pensada para integrar Autenticación de Dos Factores.
*   **Diseño Premium**: Interfaz elegante utilizando una paleta de colores curada (Mármol, Oro, Gris, Madera y Verde Bosque).
*   **Multiplataforma**: Compatible con Web, iOS, Android y Escritorio a través de la magia de Flutter.

## Requisitos Previos

Para ejecutar este proyecto, necesitarás tener instalado:

*   [Flutter SDK](https://flutter.dev/docs/get-started/install)
*   Un emulador de Android/iOS o un navegador web.
*   Un IDE compatible (VS Code, Android Studio, IntelliJ).

## Instalación y Ejecución

1.  Clona este repositorio.
2.  Ejecuta `flutter pub get` para instalar las dependencias.
3.  Ejecuta `flutter run` para iniciar la aplicación en tu dispositivo o emulador activo.
